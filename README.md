# QR Code (Tailwind)

## Overview

This is a refactored version of the QR Code Card challenge from Frontend Mentor.  
The original implementation was built using vanilla CSS and was converted into using tailwind's utility classes.

The purpose of this refactor was to practice replacing custom CSS rules with Tailwind utilities while maintaining the same visual layout and design of the card.

---

## Lab Objective

By the end of this lab, the goals were to:

- Refactor an existing project using Tailwind's utility classes
- Demonstrate the ability to leverage Tailwind documentation for efficient development
